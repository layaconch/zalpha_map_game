# 🗺️ 地图闯关游戏

基于 Web 的地图闯关游戏，通过识别地图区域来学习地理知识。

## ✨ 功能特性
- **模式**：中国省份、世界地图、欧洲地图
- **统计**：得分、正确/错误计数、正确率
- **界面**：渐变风格、响应式布局、多语言（中/英/日）

## 🚀 快速开始
```bash
npm install
npm run dev
# 构建
npm run build
# 预览
npm run preview
```

## 📁 项目结构
```
map_game/
├── index.html
├── style.css
├── game.js
├── package.json
├── vite.config.js
├── README_en.md
└── README_zh.md
```

## 🛠️ 技术栈
- 原生 JavaScript + Vite
- D3（真实地图渲染）
- SVG（抽象地图）
