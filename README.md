# 🗺️ 地图闯关游戏

一个基于 Web 的地图闯关游戏，通过识别地图区域来学习地理知识。

## ✨ 功能特性

- 🎮 三种游戏模式
  - 中国省份模式
  - 世界地图模式
  - 省内城市模式
- 📊 实时统计
  - 得分统计
  - 正确/错误计数
  - 正确率计算
- 🎨 现代化 UI
  - 渐变背景
  - 流畅动画
  - 响应式设计

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

启动开发服务器：

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

### 预览生产版本

```bash
npm run preview
```

## 🛠️ 技术栈

- 原生 JavaScript
- Vite (开发服务器和构建工具)
- SVG (地图渲染)
